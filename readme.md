# Руководство по CandleScript

CandleScript — это легкий скриптовый язык программирования, разработанный одним другом совместно со мной, предназначенный для обучения программированию, автоматизации и создания простых интерактивных сценариев. Он обладает простым синтаксисом, включающим команды ввода, вывода, условных операторов, циклов и выполнения арифметических выражений.

---

## Основные конструкции и синтаксис

### 1. Объявление и запуск скрипта
- Скрипт должен начинаться с команды candle и заканчиваться end.
- Внутри блока можно писать команды, условия, циклы и т.п.

Пример:
candle
... ваши команды ...
end

---

### 2. Команды

| Команда         | Описание                                                                                                              | Пример                                      |
|-----------------|-----------------------------------------------------------------------------------------------------------------------|----------------------------------------------|
| print         | Вывести текст или результат выражения.                                                                                | print "Hello" или print 2+3             |
| add           | Увеличить значение переменной на число. После выполнения переменная изменяется.                                       | add counter 2                             |
| sub           | Уменьшить значение переменной на число. После выполнения переменная изменяется.                                       | sub counter 1                             |
| input         | Запросить ввод пользователя, сохранить в переменную. Можно указать подсказку.                                      | input age "Ваш возраст: "                |
| calc          | Вычислить арифметическое выражение и вывести результат. Можно использовать внутри скрипта.                          | calc 5 * (2 + 3)                          |
| help          | Вывести справку по командам.                                                                                         | help                                     |

---

### 3. Объявление переменных
- Переменные создаются при присваивании через =:
  name = "Алиса"
  age = 25
  score = 0
- Значения могут быть строками (в кавычках), целыми или дробными числами.

### 4. Выражения
- Используйте calc или пишите прямо внутри print.
- Поддерживаются стандартные арифметические операции: +, -, *, /, а также скобки для группировки.

---

## Условные конструкции

### 1. if ... end
Позволяет выполнять блок команд при выполнении условия.

Синтаксис:
if <условие>
... команды ...
end

Пример:
if age >= 18
print "Доступ разрешен"
end

### 2. else
Позволяет выполнить альтернативный блок, если условие ложно.

Пример:
if age >= 18
print "Доступ разрешен"
else
print "Доступ запрещен"
end

---

## Циклы

### 1. while ... end
Повторять блок команд, пока условие истинно.

Синтаксис:
while <условие>
... команды ...
end

Пример:
while counter < 5
print counter
add counter 1
end

### 2. break
Прерывает выполнение текущего цикла.

Пример:
while true
input answer "Ваш ответ: "
if answer == "стоп"
break
end
end

---

## Операции с переменными
- Присваивание:  
  x = 10
  name = "Иван"

- Арифметические операции:  
  add score 5       // прибавить 5 к score
  sub score 2       // вычесть 2 из score

- Ввод пользователя:  
  input name "Введите имя: "

- Вывод:  
  print "Привет, " + name
  print score

Обратите внимание, что для объединения строк используйте оператор +, если он поддерживается, или внутри print через вычисление выражений.

---

## Полезные рекомендации

- Весь код внутри candle ... end интерпретируется как скрипт.
- Команды чувствительны к регистру, рекомендуется использовать строчные буквы.
- Для комментариев используйте # — всё, что после # до конца строки, будет проигнорировано.
